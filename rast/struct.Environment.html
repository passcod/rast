pub struct Environment<'e> {
    pub url: Option<Url>,
    pub raw_url: String,
    pub request_method: Method,
    pub content_length: uint,
}

Describes the environment of an HTTP request.

If url is None, it indicates the request's URL couldn't be parsed or was missing. The asterisk from RFC2616ยง5.1.2 (*) should be translated to the valid Url http://*.

The raw_url should contain the original URL as received by the server.

The content_length should reflect the value given in the headers, or 0 if not present.
