A trait for in-memory, Deque<u8>-backed streams.

The _read and _write provided methods are to be used to implement the required methods for Reader and Writer, like so:

impl Reader for SomeMemStream { fn read(&mut self, buf: &mut [u8]) -> IoResult { self._write(buf) } }

impl Writer for SomeMemStream { fn write(&mut self, buf: &[u8]) -> IoResult<()> { self._write(buf) } }

Required Methods

fn with_mode(mode: Mode) -> Self

Creates a new MemStream in either Fifo ("first in first out", the intuitive and default behaviour) or Filo ("first in last out") mode.

fn get_ref<'a>(&'a self) -> &'a T

Acquires an immutable reference to the underlying Deque.

fn unwrap(self) -> T

Unwraps this MemStream, returning the underlying Deque.

fn queue<'a>(&'a mut self) -> &'a mut T

Acquires a mutable reference to the underlying Deque.

This is used internally, and while manipulating the underlying Deque will not corrupt the MemStream, its use in external code is discouraged.

fn mode<'a>(&'a self) -> &'a Mode

Acquires an immutable reference to the Mode of this MemStream.

Provided Methods

fn new() -> Self
